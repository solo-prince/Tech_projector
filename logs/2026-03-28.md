## 🚀 Project Update - Hydroponics Automation System (HAS)

**Date:** 2026-03-28

**Progress:**
The core sensor integration phase for the nutrient solution monitoring module has reached a stable state. Successfully integrated the temperature sensor (DS18B20) and the pH sensor (Atlas Scientific EZO-pH). We have reliable local data logging to an SD card for both parameters, meeting the initial milestone for basic environmental monitoring.

**Technical Work:**
*   **Sensor Integration:** The DS18B20 temperature sensor was implemented using the OneWire protocol and the DallasTemperature library. Implemented a 1-second interval reading loop with filtering to smooth out minor fluctuations.
*   **pH Sensor Communication:** The EZO-pH module was configured for I2C communication with the Arduino Mega 2560. Developed a calibration routine based on the Atlas Scientific command set (calibration at pH 7.0 and pH 4.0).
*   **Data Logging:** Configured the SD card module (SPI communication) to store timestamped data points (Time, Temp, pH) in CSV format.

**Issues & Fixes:**
*   **pH Reading Jitter:** Initial testing showed significant instability in the pH readings, fluctuating by up to $\pm$0.3 pH units when relays for the water pumps were activated.
*   **Fix:** Traced the issue to electrical noise interference. The solution implemented involved physical separation of the sensor signal wires from high-current power lines and implementing a software-based moving average filter (averaging 10 consecutive readings) to stabilize the output data. This reduced the jitter to acceptable levels ($\pm$0.05 pH).

**Next Steps:**
*   **EC Sensor Integration:** Integrate the Atlas Scientific EZO-EC sensor into the current setup. This requires careful consideration of potential electrical noise from the EC sensor's AC excitation method.
*   **Network Module Development:** Begin development of the Wi-Fi data upload module. The plan is to use an ESP32 to receive data via serial from the Arduino Mega and publish it to an MQTT broker for dashboard visualization.
*   **Prototype Enclosure:** Start designing the final PCB layout and enclosure for the monitoring module to protect components from humidity.
